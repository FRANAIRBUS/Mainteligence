# **App Name**: Maintelligence

## Core Features:

- Incident Logging: Users can report maintenance incidents with details like location, department, priority, descriptions, and photos.
- Preventive Maintenance Scheduling: Automated scheduling of preventive maintenance tasks based on defined templates, frequencies, and checklists.
- Task Management: Maintenance technicians can view, assign, and update task statuses, track hours, parts, and add comments and photos.
- Reporting and Analytics: Generation of reports for completed tasks, including summaries, hours, materials, and checklist results.
- Admin Configuration: Administrators can configure locations, departments, equipment, users, roles, and preventive maintenance templates.
- Smart Tagging Assistant: AI powered tool to analyze the description of maintenance tasks and automatically suggest relevant tags or categories to improve organization and searchability.
- Notifications: Real-time notifications for new incidents, preventive tasks, status updates, and overdue tasks to keep users informed.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) for a professional and reliable feel, reflecting the app's core function of maintenance management.
- Background color: Light gray (#F0F4F8), provides a clean and neutral base that ensures readability and reduces eye strain.
- Accent color: Electric blue (#77B5FE), this color contrasts the deep blue providing a clear call to action.
- Headline font: 'Space Grotesk', sans-serif.
- Body font: 'Inter', sans-serif; a modern, machined, objective, neutral look that is highly legible.
- Use clean and intuitive icons for easy navigation and quick understanding of maintenance tasks.
- Prioritize a clean and organized layout, using cards and clear typography to display information efficiently.
